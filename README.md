Taller VIDAUDIO – Landing Page

Proyecto unificador del TFG · Universitat Oberta de Catalunya (UOC)

Este repositorio contiene la landing page oficial del proyecto Taller VIDAUDIO, desarrollada como parte del Trabajo Final de Grado (TFG) del grado de Comunicación de la Universitat Oberta de Catalunya (UOC).

La página sirve como punto central del proyecto unificador, cuyo objetivo es integrar varias áreas de conocimiento adquiridas durante el grado: creatividad digital, audiovisual, comunicación interactiva, narrativa, diseño y cultura digital.

Objetivo del proyecto

El propósito principal de esta landing page es:

Presentar Taller VIDAUDIO, una escuela digital de audiovisuales basada en formación accesible y microlearning.

Consolidar la identidad visual y narrativa del proyecto.

Proporcionar un espacio público para mostrar información, materiales y enlaces relacionados con la iniciativa.

Servir de base para la difusión del proyecto a través de redes sociales y plataformas externas.

Contenido del repositorio

El repositorio incluye:

index.html – Página principal de la landing page

styles.css – Hoja de estilos principal con diseño responsive

/img – Carpeta con recursos gráficos del proyecto (logo, imágenes principales, etc.)

Archivos adicionales asociados al desarrollo web

La estructura está pensada para ser simple, ligera y fácilmente escalable.

Tecnologías utilizadas

HTML

CSS

Google Fonts

Diseño adaptable básico para una correcta visualización en diferentes dispositivos

Sin frameworks externos (el proyecto está desarrollado de forma nativa) ni Javascript.

Contexto académico

Esta landing page se integra dentro del proyecto unificador del TFG, cuyo eje principal es la creación de una plataforma educativa independiente orientada al aprendizaje audiovisual. El desarrollo web forma parte del apartado práctico, complementando la memoria escrita del TFG.

Funcionalidades incluidas

Presentación general del proyecto VIDAUDIO

Enlaces a redes sociales y al canal de YouTube

Sección explicativa del enfoque de la escuela

Formulario de suscripción a newsletter (simulado mediante HTML/CSS, sin backend)

Diseño coherente con la identidad visual del proyecto basado en la identidad visual diseñada previamente.

Publicación

Esta landing page está preparada para publicarse mediante GitHub Pages, permitiendo mostrar el proyecto de forma accesible desde cualquier navegador.
